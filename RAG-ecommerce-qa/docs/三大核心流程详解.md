### 2.2 批量向量化（文档入库时）

#### 流程图

```
文档文本列表：["文本1", "文本2", "文本3", ...]
        ↓
┌─────────────────────────────────────────────────────────────┐
│ ① 检查缓存                                                 │
│    src/embedder.py 第 154-164 行                            │
│    for i, text in enumerate(texts):                         │
│        key = hashlib.md5(text.encode()).hexdigest()          │
│        if key in self.cache:                                │
│            embeddings[i] = self.cache[key]  # 有缓存直接用  │
│        else:                                                │
│            uncached_texts.append(text)  # 没缓存需要计算    │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│ ② 批量调用 BGE 模型                                        │
│    src/embedder.py 第 168-174 行                            │
│    new_embeddings = self.model.encode(                      │
│        uncached_texts,                                      │
│        batch_size=32,           # 每批 32 条                │
│        show_progress_bar=True,  # 显示进度条                │
│        normalize_embeddings=True  # L2 归一化               │
│    )                                                       │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│ ③ 存入缓存                                                 │
│    src/embedder.py 第 177-181 行                            │
│    for idx, text, emb in zip(indices, texts, embeddings):   │
│        key = hashlib.md5(text.encode()).hexdigest()          │
│        self.cache[key] = emb.tolist()                       │
│    self._save_cache()  # 保存到 JSON 文件                   │
└───────────────────────┬─────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│ ④ 返回向量数组                                              │
│    src/embedder.py 第 188 行                                │
│    return np.array(embeddings)                              │
│    → shape: (N, 512)  # N 个文本，每个 512 维               │
└─────────────────────────────────────────────────────────────┘
```

#### 代码位置

| 步骤 | 文件 | 行号 | 函数 |
|------|------|------|------|
| 缓存检查 | `src/embedder.py` | 154-164 | `embed_texts()` |
| 批量编码 | `src/embedder.py` | 168-174 | `model.encode()` |
| 缓存保存 | `src/embedder.py` | 177-181 | `_save_cache()` |
| 返回结果 | `src/embedder.py` | 188 | `embed_texts()` |

#### 关键代码

```python
# src/embedder.py 第 154-164 行 — 缓存检查
if self.use_cache:
    for i, text in enumerate(texts):
        key = self._get_cache_key(text)
        if key in self.cache:
            embeddings[i] = self.cache[key]  # 命中缓存
        else:
            uncached_indices.append(i)
            uncached_texts.append(text)  # 未命中，需要计算

# src/embedder.py 第 168-174 行 — 批量编码
if uncached_texts:
    new_embeddings = self.model.encode(
        uncached_texts,
        batch_size=self.batch_size,
        show_progress_bar=show_progress,
        normalize_embeddings=True
    )

# src/embedder.py 第 177-181 行 — 存入缓存
if self.use_cache:
    for idx, text, emb in zip(uncached_indices, uncached_texts, new_embeddings):
        key = self._get_cache_key(text)
        self.cache[key] = emb.tolist()
    self._save_cache()

# src/embedder.py 第 188 行 — 返回结果
return np.array(embeddings)
```

```
    └─ 过滤低分结果
    ↓
api.py: generate_answer_with_llm()      # ④ RAG 生成
    ├─ 构建上下文
    ├─ 构建提示词
    └─ 调用 DeepSeek API
    ↓
api.py: get_related_questions()         # ⑤ 推荐问题
    ├─ 查询同类别文档
    └─ 随机选择
    ↓
返回最终回答
```
